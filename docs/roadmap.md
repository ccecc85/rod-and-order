# Roadmap

## v0.1 Vertical Slice (Goal: "One playable shift")
**Definition of done**
- Player can complete 1 shift in 5–8 minutes
- 1 small area/cell
- 3 fault types
- 1 minigame
- Scoring and end screen
- PC build runs smoothly

**Scope**
- Top-down 2D movement
- Alarm list UI
- Interactions: inspect, test, reset, replace
- Simple state machine per machine

## v0.2 First Mobile Build
- Touch controls + UI scaling
- Performance pass
- Tutorial hints

## v0.3 Content Expansion
- 2–3 additional machines (laying head station, pinch roll, cooling conveyor)
- More fault variety
- Light narrative + characters
